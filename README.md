#  The Lost Signal

> **"Ethics is the Compass of Artificial Intelligence."**

**The Lost Signal** is an interactive web-based puzzle game developed for the **Athar Exhibition 2026**. The game simulates a telescope experience where players must locate 5 hidden stars based on signal strength to uncover a secret message.

###  How to Play
1.  **Scan the Sky:** Use the **X** and **Y** sliders to move your telescope across the galaxy.
2.  **Follow the Signal:** Watch the radar signal strength; it increases as you get closer to a star.
3.  **Capture:** Once the signal is strong (>90%), hit **SCAN** to capture the star.
4.  **The Reward:** Find all 5 stars to reveal the secret sentence and earn a personalized digital certificate.

###  Tech Stack
* **Python:** Core logic and game mechanics.
* **Streamlit:** Web framework for the interactive UI.
* **Pillow (PIL):** For dynamic image processing and certificate generation.
* **Session State:** To manage game progress, timer, and score without a database.

###  Security & Privacy
This game is designed with a **privacy-first approach**:
* No camera or microphone access required.
* No personal data is stored; all game progress is temporary (session-based).
* Safe for public exhibition use.
